## 【功能展示】（0:30 - 2:00）

### 核心功能

1. **AI 智能生成**
   - 输入描述，比如"一个日式神社"、"中世纪城堡"、"现代别墅"
   - AI 会自动生成对应的建筑结构

2. **实时 3D 预览**
   - 生成的建筑可以在 3D 视图中实时查看
   - 支持编辑视角和游戏视角切换
   - 白天/夜晚模式随意切换

3. **多格式导出**
   - NBT 结构文件（原版结构方块）
   - WorldEdit 原理图（.schem）
   - Litematica 投影（.litematic）
   - Axiom 蓝图（.bp）
   - 完整数据包（.zip）
   - 单指令方块（One Command）

4. **全版本支持**
   - 从 1.8.x 到 1.21+ 全部支持
   - 自动处理不同版本的方块名称差异

## 【使用教程】（3:00 - 5:00）

### 第一步：配置 API

⚠️ **重要提示：本软件需要配置 AI API 才能使用！**

1. 点击右上角的"设置"按钮
2. 在 API 设置中填入：
   - **API Key**：你的 API 密钥
   - **Base URL**：API 地址
   - **模型**：推荐使用 GPT-4 或同等能力的模型

**API 获取方式：**
- OpenAI 官方：https://platform.openai.com
- 国内中转站：可以在群里问
- 其他兼容 API：DeepSeek、Claude 等都可以

### 第二步：开始生成

1. 在输入框描述你想要的建筑
   - 例如："一个 10x10 的日式小屋，有樱花树"
   - 例如："中世纪风格的瞭望塔，高 20 格"
   
2. 点击发送，等待 AI 生成
3. 生成完成后可以在 3D 视图中预览

### 第三步：导出使用

1. 点击"导出"按钮
2. 选择你需要的格式
3. 选择目标 MC 版本
4. 下载文件，放到游戏里使用

## 【注意事项】（5:00 - 5:30）

1. **API 费用**
   - 本软件免费，但 AI API 调用会产生费用
   - 建议使用性价比高的 API 服务

2. **生成质量**
   - 描述越详细，生成效果越好
   - 可以指定尺寸、风格、材料等

3. **建筑大小**
   - 建议单次生成不超过 50x50x50
   - 太大的建筑可能会超时或效果不佳

4. **API 设置保存**
   - API 设置保存在浏览器本地
   - 换电脑需要重新配置
