# ✨ 魔法/巫师建筑专业知识

## 核心美学
- **气质**: 神秘、超自然、不可思议
- **特点**: 反重力、发光、扭曲、漂浮
- **氛围**: 星空、魔法阵、水晶、符文

## 建筑特征

### 巫师塔
- 高而细长，直入云霄
- 顶部有天文台或魔法实验室
- 外墙可能扭曲或倾斜
- 窗户发出神秘光芒

### 反重力元素
- 漂浮的石块
- 悬空的楼梯
- 倒挂的建筑部分
- 无支撑的平台

### 发光材料
- 发光的符文
- 魔法水晶
- 荧光植物
- 能量流动效果

### 扭曲几何
- 不可能的角度
- 螺旋上升的结构
- 空间折叠暗示
- 非欧几里得形状

## 魔法元素

### 魔法阵
地面或墙面的圆形图案，有符文和几何线条。

### 水晶
- 大型水晶簇
- 发光水晶柱
- 水晶球
- 紫水晶装饰

### 符文
墙面、地面、物品上的神秘文字。

### 魔法书架
堆满古籍的书架，书本可能漂浮。

### 炼金设备
- 蒸馏器
- 坩埚
- 药水瓶
- 神秘材料

## 空间布局

### 塔楼分层
- 底层: 入口、接待
- 中层: 图书馆、研究室
- 高层: 实验室、工作间
- 顶层: 天文台、召唤室

### 秘密空间
- 隐藏房间
- 传送门
- 次元口袋
- 地下密室

## 外部环境
- 漂浮的岛屿
- 魔法花园（发光植物）
- 守护石像
- 能量屏障
- 星空背景

## 细节装饰
- 漂浮的蜡烛
- 旋转的星象仪
- 冒泡的药水
- 闪烁的符文
- 神秘的雾气

## 不同魔法流派

| 流派 | 特点 |
|------|------|
| 元素魔法 | 火/水/土/风元素装饰 |
| 死灵魔法 | 骷髅、黑暗、灵魂火 |
| 自然魔法 | 植物、藤蔓、生命 |
| 时空魔法 | 扭曲、传送门、星空 |
| 附魔魔法 | 附魔台、经验、书籍 |
