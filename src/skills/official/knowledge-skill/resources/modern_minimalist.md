# 🏠 现代极简建筑专业知识

## 核心美学
- **理念**: Less is More（少即是多）
- **特点**: 几何感、白色主调、大面积玻璃
- **气质**: 简洁、理性、通透

## 设计原则

### 几何形体
- 以立方体、长方体为基本形
- 体块穿插、悬挑、叠加
- 避免曲线和装饰

### 材料真实
- 展示材料本身质感
- 混凝土、玻璃、钢材外露
- 不做多余装饰

### 空间流动
- 开放式平面
- 室内外空间连续
- 大面积落地窗

## 建筑特征

### 平屋顶
- 完全水平，无坡度
- 可做屋顶花园
- 边缘简洁利落

### 玻璃幕墙
- 整面玻璃墙
- 框架尽量隐藏
- 最大化采光和视野

### 悬挑结构
- 楼板或屋顶向外伸出
- 创造阴影和层次
- 展示结构力量

### 架空层
- 底层架空，柱子支撑
- 增加通透感
- 柯布西耶五原则之一

## 色彩方案

| 主色 | 用途 |
|------|------|
| 白色 | 墙面、天花 |
| 灰色 | 地面、混凝土 |
| 黑色 | 窗框、细节 |
| 原木色 | 点缀、温暖感 |

## 推荐方块

| 部位 | 方块 |
|------|------|
| 白墙 | white_concrete, smooth_quartz |
| 灰墙 | light_gray_concrete |
| 玻璃 | glass, glass_pane |
| 地面 | polished_andesite, smooth_stone |
| 木材 | stripped_oak_log, birch_planks |
| 黑色细节 | black_concrete, deepslate |

## 空间布局

### 开放平面
- 客厅、餐厅、厨房连通
- 用家具而非墙体分区
- 视线通透

### 功能分区
- 公共区: 一层，面向景观
- 私密区: 二层或后部
- 服务区: 隐蔽位置

### 室内外连接
- 大面积推拉门
- 室内地面延伸到室外
- 泳池或水景

## 细节处理
- 窗框尽量细
- 墙角锐利
- 隐藏式排水
- 嵌入式灯光
- 无把手门

## 景观设计
- 几何形草坪
- 极简水池
- 少量精选植物
- 砾石或混凝土铺地
