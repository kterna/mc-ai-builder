# 载具与机械构建专业知识

## 核心理念
载具有明确的方向性和功能性。
需要考虑：前后、上下、动力、驾驶舱。

## 船只

### 帆船
- 船体：流线型，前尖后宽
- 甲板：平坦，有舱口
- 桅杆：高耸，有横桁
- 船帆：用羊毛或旗帜

### 海盗船
- 船首有雕像
- 船尾高起（船长室）
- 大炮口
- 黑帆

### 现代船
- 金属船体
- 驾驶台在上层
- 烟囱
- 救生艇

### 构建要点
- 船底是弧形
- 用楼梯做船舷曲线
- 龙骨是核心结构

## 汽车

### 轿车
- 车身低矮
- 四个轮子
- 前后玻璃
- 车灯

### 卡车
- 驾驶室+货箱
- 轮子更大
- 可能有多轴

### 跑车
- 流线型
- 低底盘
- 大轮毂
- 尾翼

### 构建要点
- 轮子用黑色方块
- 玻璃用染色玻璃
- 车灯用发光方块

## 飞机

### 客机
- 圆柱形机身
- 后掠翼
- 尾翼（垂直+水平）
- 发动机在翼下

### 战斗机
- 尖锐机头
- 三角翼或后掠翼
- 单/双发动机
- 武器挂载

### 直升机
- 主旋翼
- 尾桨
- 起落架
- 驾驶舱玻璃大

### 构建要点
- 机翼用台阶做薄
- 发动机是圆柱
- 注意比例

## 火车

### 蒸汽机车
- 锅炉（圆柱形）
- 烟囱
- 驾驶室
- 连杆和轮子

### 现代列车
- 流线型车头
- 车厢连接
- 车窗排列
- 受电弓

### 构建要点
- 轨道用铁轨方块
- 轮子用黑色
- 车厢可以很长

## 飞船/机甲

### 宇宙飞船
- 可以不对称
- 发动机喷口
- 驾驶舱
- 武器系统

### 机甲
- 人形或动物形
- 关节可动感
- 驾驶舱在胸部或头部
- 武器手臂

### 构建要点
- 科幻感用金属方块
- 发光部件
- 细节丰富

## 推荐方块

| 用途 | 方块 |
|------|------|
| 金属 | iron_block, copper_block |
| 玻璃 | glass, tinted_glass |
| 轮子 | black_concrete, coal_block |
| 木船 | oak_planks, spruce_planks |
| 发光 | sea_lantern, glowstone |
| 细节 | 各种楼梯和台阶 |

## 通用技巧

### 流线型
- 用楼梯和台阶
- 渐变过渡
- 避免直角

### 细节
- 铆钉用按钮
- 管道用栅栏
- 天线用避雷针

### 比例
- 参考真实载具
- 可以适当夸张
- 保持协调
