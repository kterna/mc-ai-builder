# 🔱 水下/亚特兰蒂斯风格专业知识

## 核心美学
- **气质**: 神秘、古老、失落文明
- **特点**: 有机形态、发光、被水包围
- **氛围**: 深海、珊瑚、水母、古老遗迹

## 建筑特征

### 穹顶结构
水下建筑需要抵抗水压。
- 半球形或椭球形
- 玻璃或透明材料
- 可以看到外面的海洋
- 内部是干燥的生活空间

### 有机形态
模仿海洋生物的形状。
- 贝壳形
- 海螺形
- 珊瑚形
- 水母形

### 连接通道
建筑之间的透明管道。
- 玻璃隧道
- 可以看到游过的鱼
- 弧形设计

### 发光元素
深海需要人工照明。
- 生物发光
- 水晶发光
- 海晶灯
- 荧光植物

## 海洋元素

### 珊瑚
- 珊瑚礁环绕建筑
- 各种颜色的珊瑚
- 珊瑚扇装饰
- 活珊瑚需要水

### 海草森林
- 海带林
- 海草床
- 随水流摆动

### 海洋生物
- 热带鱼
- 海龟
- 海豚
- 守卫者（可作为守护者）

### 沉船残骸
- 古老的船只
- 宝箱
- 锚和链条
- 被珊瑚覆盖

| 深蓝 | 深海氛围 |

| 珊瑚色 | 珊瑚装饰 |
| 金色 | 古老文明装饰 |

## 建筑类型

### 海底神殿
- 守卫者的领地
- 海晶石结构
- 迷宫般的内部
- 宝藏室

### 穹顶城市
- 巨大的玻璃穹顶
- 内部有建筑群
- 人工阳光
- 花园和广场

### 珊瑚宫殿
- 由珊瑚构成
- 有机形态
- 色彩斑斓
- 与珊瑚礁融为一体

### 沉没遗迹
- 古老文明的残骸
- 被海草覆盖
- 部分坍塌
- 神秘符文

## 布局原则

### 垂直发展
水下空间向上下延伸。

### 有机布局
避免直线，模仿自然形态。

### 光源分布
确保各处都有照明。

### 水密性
建筑内部需要隔绝海水。
