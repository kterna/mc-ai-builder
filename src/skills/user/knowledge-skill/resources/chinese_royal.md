# 🏛️ 中式皇家建筑专业知识

## 核心美学
- **气质**: 宏大、威严、等级森严
- **色彩**: 红墙黄瓦（皇家专用配色）
- **布局**: 严格中轴对称，院落递进

## 建筑特征

### 三层台基（须弥座）
皇家建筑必须建在高台基上，彰显地位。最高等级用三层汉白玉台基，每层设栏杆，四面设台阶。

### 朱红大柱
直径粗大的圆柱，通体朱红色，柱头有金色装饰。柱子排列整齐，形成柱廊。

### 斗拱
柱顶与屋檐之间的木构件，层层叠叠向外挑出。既承重又装饰，彩绘华丽。

### 屋顶形式（等级从高到低）
1. **重檐庑殿顶**: 最高等级，四面坡+重檐，如太和殿
2. **重檐歇山顶**: 次高，四面坡+两侧山花
3. **单檐庑殿顶**: 四面坡单层
4. **单檐歇山顶**: 一般殿宇
5. **悬山顶/硬山顶**: 普通建筑

### 脊兽
屋脊上的装饰兽，数量代表等级。太和殿有10个（最高），依次为龙、凤、狮子、天马、海马、狻猊、押鱼、獬豸、斗牛、行什。

## 色彩规范

| 颜色 | 使用场合 |
|------|----------|
| 黄色 | 皇帝专用，琉璃瓦 |

| 白色 | 台基、栏杆（汉白玉） |
| 金色 | 装饰点缀 |

## 布局原则

### 中轴对称
所有主要建筑沿中轴线排列，左右严格对称。

### 院落递进
- 外朝: 举行大典（太和殿、中和殿、保和殿）
- 内廷: 皇帝起居（乾清宫、交泰殿、坤宁宫）
- 后苑: 御花园

### 开间规制
- 11开间: 最高等级（太和殿）
- 9开间: 次高
- 5-7开间: 一般殿宇
- 3开间: 小型建筑

## 装饰元素
- 龙纹: 皇帝象征，无处不在
- 铜狮/铜鹤: 殿前陈设
- 日晷/嘉量: 象征皇权
- 宫灯: 大型红色灯笼
