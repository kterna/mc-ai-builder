# 🌃 赛博朋克风格专业知识

## 核心美学
- **理念**: High Tech, Low Life（高科技，低生活）
- **特点**: 霓虹灯、黑暗、拥挤、科技与颓废并存
- **氛围**: 雨夜、烟雾、全息投影、电子音乐

## 视觉特征

### 霓虹灯
无处不在的发光招牌。
- 粉色、蓝色、紫色为主
- 汉字/日文招牌
- 闪烁效果
- 反射在湿润地面

### 黑暗基调
- 永远是夜晚
- 阴影和光的强烈对比
- 烟雾弥漫
- 雨水不断

### 高密度建筑
- 摩天大楼密集
- 空中走廊连接
- 层层叠叠
- 狭窄街道

### 科技元素
- 全息广告
- 无人机
- 机械义肢
- 电缆管线外露

## 建筑特征

### 摩天大楼
- 巨型企业总部
- 玻璃幕墙
- 顶部有直升机停机坪
- 巨型广告屏

### 贫民窟
- 临时搭建
- 材料混杂
- 电线乱拉
- 涂鸦覆盖

### 街道层
- 狭窄拥挤
- 小摊贩
- 霓虹招牌
- 蒸汽排放口

### 空中层
- 天桥连接
- 轨道交通
- 广告牌
- 无人机航线

## 氛围营造

### 雨夜效果
- 湿润的地面（水）
- 反光效果
- 雾气弥漫

### 光污染
- 霓虹灯到处都是
- 没有真正的黑暗
- 颜色混杂

### 拥挤感
- 建筑紧密
- 街道狭窄
- 层层叠加
