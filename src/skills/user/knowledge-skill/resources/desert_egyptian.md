# 🐪 沙漠/古埃及风格专业知识

## 核心美学
- **气质**: 宏大、神秘、永恒、神圣
- **特点**: 几何形体、巨大尺度、象形文字
- **氛围**: 烈日、黄沙、尼罗河、法老

## 建筑特征

### 金字塔
法老的陵墓，追求永恒。
- 正四棱锥形
- 底边精确朝向东南西北
- 内部有墓室和通道
- 表面原本覆盖白色石灰岩

### 神庙
供奉神明的场所。
- 轴线布局
- 从入口到圣所逐渐变暗变窄
- 巨大的柱厅
- 墙面满是浮雕和象形文字

### 方尖碑
单块巨石雕刻的纪念碑。
- 四面锥形
- 顶部镀金
- 刻有法老功绩
- 成对放置在神庙入口

### 塔门（Pylon）
神庙入口的巨大门楼。
- 两个梯形塔楼
- 中间是入口
- 表面有浮雕
- 顶部有旗杆槽

## 柱式

### 纸莎草柱
模仿纸莎草束。
- 柱头如盛开的花
- 或如未开的花苞
- 柱身有凹槽

### 莲花柱
模仿莲花。
- 柱头如莲花
- 象征上埃及

### 棕榈柱
模仿棕榈树。
- 柱头如棕榈叶
- 较为少见

## 布局原则

### 轴线对称
所有建筑沿中轴线排列。

### 递进空间
入口 → 庭院 → 柱厅 → 圣所
- 空间逐渐变小
- 光线逐渐变暗
- 地面逐渐升高
- 天花板逐渐降低

### 巨大尺度
一切都是巨大的，彰显法老权力。
