# ☁️ 浮空岛/天空建筑专业知识

## 核心美学
- **气质**: 神圣、轻盈、超凡脱俗
- **特点**: 反重力、云雾、光芒、纯净
- **氛围**: 天堂、仙境、梦幻

## 浮空岛结构

### 岛屿基座
浮空岛必须有底部！
- 倒锥形或不规则岩石
- 底部有瀑布流下
- 根系或藤蔓悬垂
- 发光的能量核心

### 岛屿类型
- **自然岛**: 草地、树木、小溪
- **神殿岛**: 白色建筑、金色装饰
- **废墟岛**: 残破建筑、藤蔓覆盖
- **水晶岛**: 水晶构成、发光

### 连接方式
- 链条桥
- 彩虹桥
- 传送门
- 漂浮阶梯

## 建筑特征

### 轻盈感
- 细长的柱子
- 大量镂空
- 透明材料
- 向上延伸的尖顶

### 白色主调
- 白色石材
- 金色装饰
- 蓝色点缀
- 发光元素

### 对称与秩序
- 严格对称布局
- 几何图案
- 放射状设计
- 中心焦点

## 天空元素

### 云层
用白色羊毛或混凝土模拟云朵，可以是：
- 平台状云层
- 漂浮的云朵
- 云梯

### 瀑布
从浮空岛边缘流下的水流，消失在云层中。

### 彩虹
连接不同岛屿的彩色拱桥。

### 光柱
从天而降或从地升起的光束。

## 建筑类型

### 天使圣殿
- 纯白建筑
- 金色穹顶
- 羽翼装饰
- 圣光照耀

### 天空城市
- 多个浮空岛
- 桥梁连接
- 不同功能区
- 中央主岛

### 云端花园
- 漂浮的花圃
- 喷泉和水池
- 发光植物
- 休憩亭台

## 布局原则

### 层次分明
- 最高: 神殿/宫殿
- 中层: 居住/花园
- 底层: 入口/码头

### 视觉焦点
中央有标志性建筑或雕塑。

### 开放空间
大量露台和观景平台。

## 细节装饰
- 漂浮的灯笼
- 飘动的旗帜
- 天使雕像
- 星星点缀
- 光环效果
