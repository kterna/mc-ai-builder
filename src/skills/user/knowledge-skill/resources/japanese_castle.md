# 🏯 日式城堡专业知识

## 核心美学
- **气质**: 威严、雄伟、军事防御
- **色彩**: 白色墙面 + 灰色瓦顶 + 金色装饰
- **特点**: 层层叠叠，向上收缩

## 建筑特征

### 天守阁（Tenshu）
城堡的核心建筑，多层塔楼结构。
- 3-7层不等
- 每层向上逐渐缩小
- 最顶层有金色装饰

### 石垣（石墙）
巨大的石头堆砌的城墙基座。
- **切込接**: 石块精密切割，严丝合缝
- **打込接**: 石块粗加工，缝隙用小石填充
- **野面積**: 自然石堆砌
- 墙面向内倾斜（扇形坡度）

### 屋顶
- 多层屋顶，每层独立
- 入母屋造（歇山顶）最常见
- 屋脊有鯱（虎头鱼身装饰）

### 破风（山花）
屋顶的三角形装饰面。
- **千鸟破风**: 三角形
- **唐破风**: 弧形
- 常有金色装饰

## 防御设施

### 狭间（射击孔）
墙上的小孔，用于射箭或射击。形状有圆形、方形、三角形。

### 石落
墙体突出部分，底部开口，可向下投掷石块或热水。

### 堀（护城河）
环绕城堡的水沟，分内堀和外堀。

### 虎口
城门入口的曲折通道，迫使敌人暴露侧面。

## 布局原则

### 曲轮（郭）
城堡由多个区域组成：
- **本丸**: 核心区，天守阁所在
- **二之丸**: 第二层防御
- **三之丸**: 外围区域

### 层层防御
敌人必须突破多道城门和城墙才能到达天守阁。

### 高低差
利用地形，建在山上或人工堆高的台地上。

## 著名城堡参考
- **姫路城**: 白鹭城，最完整的现存城堡
- **大阪城**: 金色装饰华丽
- **熊本城**: 石垣技术最高
- **松本城**: 黑色外观，平地城堡

## 装饰元素
- 鯱: 屋脊两端的虎头鱼身
- 金箔: 破风和细部装饰
- 家纹: 城主家族徽章
- 灯笼: 石灯笼或金属灯笼
