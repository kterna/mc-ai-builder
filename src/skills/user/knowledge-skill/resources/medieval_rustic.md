# 🛖 中世纪乡村风格专业知识

## 核心美学
- **气质**: 温馨、粗犷、朴实
- **特点**: 不对称、有机感、岁月痕迹
- **氛围**: 炊烟袅袅、灯火温暖

## 建筑特征

### 半木结构（Tudor Style）
木框架外露，框架间填充灰泥或砖石。木材呈深色，填充物呈浅色，形成强烈对比。框架常有斜撑，增加结构稳定性和装饰效果。

### 石基
建筑底部1-2层用石头砌筑，防潮防腐。常用不规则的毛石或鹅卵石。

### 陡峭屋顶
坡度45-60度，便于排水排雪。材料为茅草、木瓦或石板瓦。屋檐悬挑保护墙面。

### 烟囱
必备元素！通常位于山墙一侧，用石头或砖砌筑。烟囱口略微收窄，顶部有遮雨帽。

### 门廊
入口处有小型遮雨棚或门廊，用木柱支撑。门前常有石阶。

## 窗户特点
- 小而少（保暖需要）
- 带木质百叶窗
- 窗台放花盆
- 窗框用深色木材

## 内部空间

### 壁炉区
乡村小屋的核心，用于取暖和烹饪。壁炉用石头砌筑，上方有烟道。炉边常设座椅。

### 典型家具
- 粗木餐桌和长凳
- 简易木床，铺稻草或羊毛
- 储物箱和木桶
- 悬挂的锅具和食材

## 外部环境
- 篱笆围成的小院
- 菜园和花圃
- 水井或水泵
- 柴堆和工具棚
- 鸡舍或猪圈

## 做旧效果
- 墙面长青苔和藤蔓
- 石头混入苔石
- 木材颜色不均匀
- 屋顶有缺损和修补痕迹

## 变体类型

| 类型 | 特点 |
|------|------|
| 酒馆 | 更大，有招牌，多窗户 |
| 铁匠铺 | 外部有熔炉和铁砧 |
| 面包房 | 大烟囱，外部有小麦装饰 |
| 磨坊 | 带风车或水车 |
| 教堂 | 有钟楼，彩色玻璃窗 |
