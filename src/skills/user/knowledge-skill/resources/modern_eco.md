# 🍃 现代生态建筑专业知识

## 核心美学
- **理念**: 建筑与自然共生
- **特点**: 大量植物、自然采光、可持续
- **氛围**: 清新、健康、未来感

## 设计原则

### 垂直绿化
建筑立面覆盖植物。
- 攀爬植物
- 种植槽
- 绿墙系统
- 阳台花园

### 屋顶花园
屋顶变成花园。
- 草坪
- 灌木
- 小树
- 菜园

### 自然采光
最大化利用自然光。
- 大面积玻璃
- 天窗
- 中庭
- 光导管

### 自然通风
减少空调依赖。
- 可开启窗户
- 通风井
- 风塔
- 双层立面

## 建筑特征

### 阶梯式退台
每层向后退，形成露台。
- 每层都有户外空间
- 种植植物
- 私密性好

### 中庭
建筑中央的开放空间。
- 贯穿多层
- 自然采光
- 室内花园
- 社交空间

### 双层玻璃幕墙
两层玻璃之间有空腔。
- 隔热
- 通风
- 种植植物

### 遮阳系统
控制阳光进入。
- 百叶
- 穿孔板
- 植物遮阳
- 智能控制
