# 🪓 北欧/维京风格专业知识

## 核心美学
- **气质**: 粗犷、勇猛、适应严寒
- **特点**: 木结构、龙图腾、长屋形式
- **氛围**: 峡湾、雪山、极光、篝火

## 建筑特征

### 长屋（Longhouse）
维京人的主要居所。
- 长方形，长度可达30-50米
- 弧形屋顶（像倒扣的船）
- 中央有火塘
- 两侧是睡眠和储物区

### 木板教堂（Stave Church）
基督教化后的建筑。
- 多层屋顶，层层叠叠
- 龙头装饰屋脊
- 全木结构
- 雕刻精美

### 大厅（Mead Hall）
首领的宴会厅。
- 比长屋更高大
- 有高座（首领座位）
- 墙上挂盾牌和武器
- 举行宴会和议事

## 结构要素

### 龙头装饰
屋脊两端的龙头雕刻。
- 张嘴露齿
- 向外伸出
- 驱邪避凶

### 木雕
门框、柱子上的复杂雕刻。
- 缠绕的藤蔓图案
- 神话故事
- 动物图案

### 草皮屋顶
用草皮覆盖的屋顶。
- 保温效果好
- 与环境融合
- 需要定期维护

### 火塘
长屋中央的火坑。
- 取暖和烹饪
- 烟从屋顶孔洞排出
- 社交中心

## 装饰元素

### 盾牌
墙上悬挂的圆盾。
- 各种颜色
- 有图案
- 排列整齐

### 武器
- 战斧
- 长剑
- 长矛
- 弓箭

### 兽皮
地面和墙上的动物皮毛。

### 符文
神秘的北欧文字。
- 刻在石头上
- 刻在木头上
- 有魔法含义

### 号角
饮酒用的牛角杯。
