# 🌾 乡村农场风格专业知识

## 核心美学
- **气质**: 朴实、温暖、勤劳、自给自足
- **特点**: 实用主义、宽大空间、与土地结合
- **氛围**: 田野、牲畜、丰收、宁静

## 建筑类型

### 农舍（Farmhouse）
农民居住的房屋。
- 两层结构
- 大厨房（家庭中心）
- 前廊
- 烟囱

### 谷仓（Barn）
储存粮食和设备。
- 高大宽敞
- 大门可进马车
- 阁楼储存干草
- 红色或原木色

### 马厩（Stable）
饲养马匹。
- 一排马厩
- 每个隔间有门
- 通风良好
- 附近有草场

### 风车（Windmill）
磨面或抽水。
- 高塔结构
- 四叶或多叶风帆
- 可旋转的顶部
- 内部有磨盘

### 筒仓（Silo）
储存谷物。
- 圆柱形
- 金属或砖石
- 顶部有圆锥盖
- 底部有出料口

## 建筑特征

### 大坡屋顶
- 便于排水排雪
- 阁楼空间大
- 可储存干草

### 宽大门廊
- 遮阳避雨
- 休息场所
- 社交空间

### 石基木墙
- 底部石头防潮
- 上部木板墙
- 经济实用

### 多烟囱
- 厨房烟囱
- 壁炉烟囱
- 取暖需要

## 农场设施

### 鸡舍
- 小型木屋
- 有栖息杆
- 产蛋箱
- 小门进出

### 猪圈
- 围栏围起
- 有遮蔽棚
- 泥地
- 食槽水槽

### 牛棚
- 较大空间
- 分隔栏
- 挤奶区
- 饲料槽

### 菜园
- 整齐的菜畦
- 篱笆围护
- 稻草人
- 堆肥区

## 田园元素

### 田野
- 小麦田
- 玉米地
- 向日葵
- 南瓜地

### 果园
- 苹果树
- 樱桃树
- 整齐排列

### 牧场
- 大片草地
- 木栅栏
- 饮水槽
- 遮阳棚

### 池塘
- 鸭子
- 垂柳
- 钓鱼台

## 装饰细节
- 马车
- 农具（锄头、镰刀）
- 木桶
- 草垛
- 风向标
- 邮箱
- 摇椅

## 色彩方案

| 颜色 | 用途 |
|------|------|
| 红色 | 谷仓传统色 |
| 白色 | 农舍、围栏 |
| 棕色 | 木材 |
| 绿色 | 田野、草地 |
| 金色 | 小麦、干草 |

## 布局原则
- 农舍在中心
- 谷仓在旁边
- 牲畜区在下风向
- 菜园靠近厨房
- 果园在外围
