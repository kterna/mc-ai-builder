# ⚙️ 蒸汽朋克/工业风格专业知识

## 核心美学
- **时代**: 维多利亚时代 + 蒸汽动力幻想
- **特点**: 齿轮、管道、铜质、蒸汽、机械
- **氛围**: 工厂轰鸣、蒸汽弥漫、黄铜光泽

## 视觉特征

### 机械元素
- 巨大的齿轮
- 外露的管道
- 活塞和连杆
- 压力表和阀门

### 材质
- 黄铜/铜色金属
- 铆钉连接
- 锈迹斑斑的铁
- 砖石结构

### 蒸汽效果
- 排气管冒烟
- 蒸汽阀门
- 冷凝水滴
- 雾气弥漫

## 建筑特征

### 工厂
- 高大的烟囱
- 锯齿形屋顶（采光）
- 大型窗户
- 起重机和吊车

### 飞艇坞
- 巨大的拱形结构
- 停靠平台
- 系泊塔
- 加气站

### 钟楼
- 巨大的钟面
- 外露的齿轮机构
- 铜质装饰
- 蒸汽动力

### 机械城市
- 多层结构
- 管道网络
- 电梯和升降机
- 蒸汽火车轨道

## 建筑细节

### 铆钉
金属板连接处的装饰性铆钉。

### 烟囱
- 高耸的圆柱形
- 顶部有遮雨帽
- 冒出蒸汽/烟雾

### 窗户
- 工业风格大窗
- 铁框架
- 可能有破损

### 门
- 厚重的铁门
- 铆钉装饰
- 圆形舷窗

## 交通工具

### 蒸汽火车
- 蒸汽机车头
- 大烟囱
- 连杆和轮子

### 飞艇
- 气囊
- 吊舱
- 螺旋桨
- 舵和帆

### 机械步行器
- 蜘蛛形或双足
- 蒸汽动力
- 驾驶舱
