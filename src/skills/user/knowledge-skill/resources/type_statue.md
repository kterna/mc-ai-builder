# 雕像与有机体构建专业知识

## 核心理念
这不是建筑！不要造房间、墙壁或屋顶。
这是有机雕塑，需要用体素模拟曲面和生物形态。

## 构建方法

### 体素化思维
- 把复杂形状分解为简单几何体
- 球体、椭球体、圆柱体组合
- 逐层构建，像3D打印

### 轮廓优先
- 先确定整体轮廓
- 从大形到小形
- 最后添加细节

### 对称利用
- 人物和动物通常左右对称
- 先做一半，再镜像
- 注意动态姿势的不对称

## 人物雕像

### 比例参考
- 头部：约占身高1/7-1/8
- 肩宽：约2个头宽
- 手臂：垂下时指尖到大腿中部
- 腿长：约占身高1/2

### 构建顺序
1. 躯干（核心）
2. 头部
3. 四肢
4. 手脚
5. 面部细节
6. 服装/装备

### 姿势
- 站立：最简单
- 坐姿：需要座椅
- 动态：需要支撑结构

## 动物/生物

### 四足动物
- 身体是椭球体
- 四肢是圆柱体
- 头部单独处理
- 尾巴是锥形

### 鸟类
- 身体是卵形
- 翅膀是扁平体
- 喙是锥形
- 注意羽毛层次

### 龙/怪物
- 可以夸张比例
- 翅膀要大
- 爪子要锋利
- 鳞片用楼梯方块

## 推荐方块

| 用途 | 方块 |
|------|------|
| 皮肤 | smooth_quartz, sandstone |
| 石像 | stone, andesite |
| 铜像 | copper_block, oxidized_copper |
| 金像 | gold_block |
| 黑色 | blackstone, obsidian |
| 细节 | 各种楼梯和台阶 |

## 细节技巧

### 面部
- 眼睛用黑色方块
- 鼻子用台阶
- 嘴用台阶或缝隙

### 手指
- 小型雕像可省略
- 大型雕像用台阶
- 或简化为手套形

### 衣褶
- 用楼梯表现
- 注意重力方向
- 动态褶皱

### 毛发/羽毛
- 用羊毛或树叶
- 层层叠加
- 注意生长方向

## 底座设计
- 雕像需要底座
- 底座比雕像宽
- 可以有铭文
- 材质与雕像协调

## 尺寸建议
- 小型：10-20格高
- 中型：30-50格高
- 大型：100格以上
- 越大越需要简化细节
